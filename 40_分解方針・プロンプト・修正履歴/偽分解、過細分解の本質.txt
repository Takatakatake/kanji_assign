■ 偽分解・過細分解の本質

【偽分解の目的】
偽分解（学習者版だけの分解）は、注釈ルビを振るためのもの。
狙いは「語全体の意味を、構成片の意味から学習者がクリアに復元できる」ようにすること。


【判定の核心】
分解してよいかは、ただ一点——
　〔各片の意味を足し合わせて、語全体の意味がクリアに立ち上がるか？〕

語根とみなしてよいのは、正式語根と、非常に汎用性の高い断片だけ。
ただし logi / log / ism / elektr のような「非常に」強い語根については、
相方が汎用性は低くても、相方がその語の中で「具体的・明確な意味」を担い、
全体がクリアに立ち上がるなら、強い語根の存在感がそれを引き受けて分解してよい。

逆に、強い語根があっても、相方の意味が曖昧・抽象的で全体がクリアに
立ち上がらないなら、分解しない。（迷えば一塊が原則）


【◯ 分解してよい例】 fraz/e/o/logi/o（句法・語法 ＝ phraseologio）
　・fraz/o（句・文、{Ｂ}）＝具体的で明確。
　・logi（学）＝非常に強い。
　・「句＋学」で〈句の研究〉がクリアに立ち上がる → 分解する。


【× 分解しない例】 elektrofor/o（電気盆 ＝ electrophorus）
　・elektr/o（電気）＝非常に強い。
　・しかし for は phoros＝「運ぶもの（carrier）」で抽象的。
　　（fosfor/o＝リン、metafor/o＝隠喩、semafor/o＝腕木信号 と同じ -foro 系。
　　　なお「釜」は別語根 forn/o（n付き）！ for と forn は別物。）
　・「電気＋運ぶもの」では〈電気盆〉がクリアに立ち上がらない → 分解しない。

→ 同じ「非常に強い elektr/o」を持っていても、相方 for の意味が抽象的なので分解しない。
　決め手は〈強い語根の有無〉ではなく、〈相方が具体的な意味を担い、全体がクリアに立ち上がるか〉。


【× 分解しない例（その他の典型）】
　・tetralogi/o（四部作）… tetra＝四 は明確でも、ここの logi は「学」でなく
　　「作品の連なり」。tetra/logi＝「四＋学」と読めて〈四部作〉に結びつかず弱い → 分解しない。
　・doksologi/o（頌栄）… dokso（栄光/見解）が曖昧 → 分解しない。
　・kaligraf/o（書道）… kali＝カリウム（≠美）の意味ギャップ → 分解しない。


【もう一つの判定軸 ── 「エスペラント化の努力」】
語全体の意味が注釈からクリアに立ち上がるなら、言うまでもなく分解する（これが最優先）。
　それに加えて——
　構成片が「認識可能なエスペラント形態素」で組み立てられ、
　単語をエスペラント風に寄せようとした“努力”が感じられる語は、
　たとえ語全体の意味が多少立ち切らなくても、分解してよい。
　構成片を露出させること自体に学習補助の価値があるからである。（主に学名・造語が対象。）


【◯ 努力が感じられる → 分解する】
　・akr/o/cefal/o（鳥の属 Acrocephalus）
　　… akr（尖）＋ cefal（頭）はともに認識可能なエスペラント形態素。
　　　全体は「鳥」とまでは立ち上がらないが、akr・cefal を組んで作った“努力”が明確 → 分解する。
　・mega/lit/o（巨石）… mega（大）＋ lit（石）。既に分解されている mono/lit/o（一枚岩）に並ぶ。
　・hemat/o/pod/o（血足＝ミヤコドリ）、akant/o/cefal/o（鉤頭虫）なども同様。


【× 努力が感じられない（opaque な“ただの借音”）→ 一塊】
決め手は〈第1要素が認識可能なエスペラント形態素か、それとも辞書のどこにも露出しない“ただの借音”か〉。
　・orikter/o/pod/o（ツチブタ）… orikter（掘る者）が他に一切露出しない opaque な借音 → 一塊。
　・kope/pod/o（橈脚類）… kope（櫂）が露出しない借音 → 一塊。
　・apolog/o（寓話）… apo（〜から／off）が抽象的な方向接頭辞で意味を決めきれず、この族にしか現れない → 一塊。
　（強い語根 log〔言・談〕を相方に持っても、第1要素 apo が抽象的・非決定なら全体は立たない＝elektrofor 型。）


【混同しないこと ── 「努力」型と、別の“立たない”型】
部品が一見それらしくても、次の型は意味が立たない／誤誘導するため、努力型と区別して一塊にする。
　・false friend 型：露出する片の意味が、その語では誤り。
　　　braki＝腕（≠ 短 brachy）、karp＝コイ（≠ 果）、himen＝処女膜（≠ 膜 membrane）。
　・sense-shift 型：第2要素を、その露出義から外れた意味へ読み替える必要がある。
　　　alotrop の trop＝転回 →「形」へ読み替え（同素体）。tetralogi の logi＝学 →「作品の連なり」。
　　　（alotrop は「エスペラント化の努力」も感じられない、とも判断できる。）


【過細分解は「おまけ」】
過細分解は、偽分解で露出した片が「同綴りの標準語根」と衝突するのを、
注釈生成時に避けるための機械的な裏処理。あくまで、偽分解された語が対象である。
標準（正式）語根に過細分解が施されることは絶対にない。

　（見出しは通常の分解形のまま保ち、letter 単位の形を別フィールドに置く。
　　例：for を露出すると for〔〜離れて〕と衝突 → 一応 f/o/r 化。）
本質的には重要でない。偽分解の判断を過細分解の都合で曲げてはいけない。
o-chunk化なども、あくまで、意味的分解が圧倒的基準としてある。

　　★ 優先順位は 偽分解 ≫ 過細分解。







本来、PIVに正式語根として認められている語根のみによる分解であれば、「偽分解」という印すら適切ではないかもしれない。しかし、その語がPEJVOで分解されなかった背景には、語根の汎用性など、それなりの理由がありそうに思われる。その情報まで消してしまうのは忍びないため、##偽分解(PIV正式分解)と表記することとした。理論上、現在も最初の４４１０４行はPEJVO由来、それ以降はPIV由来のままである。したがって、偽分解・過細分解の方針を遵守すれば、一意的な偽分解・過細分解が再現可能なはずである！
